# 💻 Project Olympus - Programming Plan

## Programming Overview

The goal of the programming system is to create the core gameplay mechanics of Project Olympus.

The first prototype will focus on:

- Player movement
- Combat
- Enemy behavior
- Character stats
- Skills
- Inventory

---

# 📁 Script Structure

Unity scripts will be organized like this:

---

# 🧍 Player Systems

## PlayerController

Controls:

- Movement
- Running
- Jumping
- Camera movement

---

## PlayerCombat

Controls:

- Basic attacks
- Heavy attacks
- Skills
- Damage effects

---

## PlayerStats

Stores:

- Health
- Attack
- Defense
- Level
- Experience

---

# ⚔️ Combat Systems

## Health System

Handles:

- Taking damage
- Healing
- Death
- Respawning

---

## Damage System

Calculates:

- Physical damage
- Element damage
- Critical hits
- Damage resistance

---

## Skill System

Controls:

- Skill activation
- Cooldowns
- Energy usage
- Effects

---

# 👹 Enemy Systems

## Enemy AI

Controls:

- Movement
- Attacking
- Chasing player
- Retreating

---

## Boss System

Creates unique boss behaviors:

Examples:

- Multiple attack phases
- Special abilities
- Weak points

---

# 🎒 Inventory System

Handles:

- Items
- Equipment
- Materials
- Rewards

---

# 💾 Save System

Stores:

- Player progress
- Character levels
- Equipment
- Story progress

---

# Coding Philosophy

The code should be:

- Easy to understand
- Organized
- Expandable
- Beginner friendly

The goal is not only to make a game, but to create a foundation that can grow into a complete RPG.
